**Services.tsx:**

Replace the `space-y-10` container with `divide-y divide-text-tertiary/20` for ruled lines between items. Add `group` and `cursor-default` to each item wrapper, and `group-hover:text-text-primary transition-colors duration-200` to each description paragraph.

Current:
```jsx
<div className="space-y-10">
  {services.map((service) => (
    <div key={service.id}>
      <h3 className="text-xl mb-2">{service.title}</h3>
      <p className="text-text-secondary">{service.description}</p>
    </div>
  ))}
</div>
```

Replace with:
```jsx
<div className="divide-y divide-text-tertiary/20">
  {services.map((service) => (
    <div key={service.id} className="py-8 first:pt-0 last:pb-0 group cursor-default">
      <h3 className="text-xl mb-2">{service.title}</h3>
      <p className="text-text-secondary transition-colors duration-200 group-hover:text-text-primary">
        {service.description}
      </p>
    </div>
  ))}
</div>
```

Key details:
- `divide-y divide-text-tertiary/20` creates thin horizontal rules between each item (Tailwind adds `border-top` to every child except the first)
- `py-8` gives consistent vertical padding to each item (replacing the `space-y-10` gap)
- `first:pt-0 last:pb-0` removes extra padding at top/bottom of the list
- `group cursor-default` enables group-hover without implying the items are clickable
- `group-hover:text-text-primary` brightens description text on hover per locked decision
- `transition-colors duration-200` smooths the hover color change

**About.tsx — photo mask-image fade:**

Edit the photo wrapper `<div>` in About.tsx. Remove `rounded-lg` and `bg-text-tertiary/10` classes (they conflict with the fade effect). Add inline style for `maskImage` and `WebkitMaskImage` with a radial gradient that fades edges into the background.

Current photo wrapper:
```jsx
<div className="w-48 h-48 md:w-full md:h-auto rounded-lg overflow-hidden bg-text-tertiary/10">
```

Replace with:
```jsx
<div
  className="w-48 h-48 md:w-full md:h-auto overflow-hidden"
  style={{
    maskImage: 'radial-gradient(ellipse 85% 85% at 50% 40%, black 40%, transparent 80%)',
    WebkitMaskImage: 'radial-gradient(ellipse 85% 85% at 50% 40%, black 40%, transparent 80%)',
  }}
>
```

Key details:
- `radial-gradient(ellipse 85% 85% at 50% 40%, ...)` — the opaque center is slightly above-center (good for portrait photos where face is in upper portion)
- `black 40%` — center 40% is fully opaque
- `transparent 80%` — edges fade to transparent between 40% and 80%
- Both `maskImage` and `WebkitMaskImage` are required for cross-browser support (Safari needs -webkit prefix)
- Removed `rounded-lg` — rounded corners conflict with the radial fade aesthetic
- Removed `bg-text-tertiary/10` — the background placeholder color shows through the transparent mask edges, which looks wrong
